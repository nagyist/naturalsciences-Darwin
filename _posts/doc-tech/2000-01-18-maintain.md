Upgrade Darwin :
================

For the web interface , just do :
{% highlight bash %}
cd /var/www/web
git pull

# reset the cache
php symfony cc
sudo /etc/init.d/apache reload
{% endhighlight %}

For the database do :

{% highlight bash %}
cd web/data/db/ # DB directory on your Database server
sudo -u postgres make upgrade
{% endhighlight %}
